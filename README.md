# UniSphere

## 📌 Overview
UniSphere is a web-based Learning Management System (LMS) built using Flask. It provides a simple platform where users can manage academic activities, interact with course-related content, and handle user authentication.

## 🚀 Features
- User registration and login system
- Secure authentication and session management
- Course or content management (basic)
- Database integration for storing user data

## 🛠️ Tech Stack
- Backend: Flask
- Authentication: Flask-Login
- Database: Flask-SQLAlchemy
- Security: Werkzeug
